# Ethio-Exam AI

Ethio-Exam AI is a comprehensive mobile application designed for Ethiopian high school students preparing for University Entrance Exams (Grade 12).

## Core Features

- **AI Lens (Camera Module)**: Scan questions and get instant AI-extracted text.
- **Learning Dashboard**: Track "Exam Readiness," daily streaks, and review weak subjects.
- **Exam Engine**: Solve questions with LaTeX formula support and step-by-step AI explanations referenced to MOE textbooks.
- **Categorization System**: Drill down by Subject -> Grade -> Chapter.
- **Study Planner**: Calendar-based view with drag-and-drop scheduling and push notifications.
- **Account Deletion Logic**: Accounts are automatically marked for deletion after the national exam date.

## Tech Stack

- **Frontend**: Flutter
- **Backend**: NestJS / PostgreSQL
- **OCR**: Google ML Kit
- **Math Rendering**: LaTeX

## Visual Style

- **Primary colors**: Academic Blue (#1A237E) and Growth Green (#2E7D32).
- **Vibe**: Modern, clean, and accessible for various screen sizes.
- **High contrast** text for readability.
